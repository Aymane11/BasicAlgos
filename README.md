## Usage
#### C++ :
Add the `.h` file desired to the includes of your file to use the algorithm
#### Python :
Add `from <filename> import *` to your file

### Example
##### C++ :
```cpp
#include "bubblesort.h"
//rest of code
```
##### Python :
```python
from fibonacci import *
#rest of code
```
